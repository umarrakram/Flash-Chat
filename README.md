# Flash-Chat-iOS13

Real-time chat app using Firebase Auth and Firestore, with custom UITableView cells and MVC architecture.

> **Attribution:** Follow-along project from [Angela Yu's Complete iOS App Development Bootcamp](https://www.appbrewery.co/). Original starter code by [App Brewery](https://github.com/appbrewery/Flash-Chat-iOS13).

## What I practiced
- Firebase Authentication (email/password sign-up & login)
- Firebase Firestore (send/receive messages live, sorted by date)
- Custom UITableViewCell with XIB
- UITableView data sources and delegates
- Navigation stack and segues
- MVC architecture with constants file
